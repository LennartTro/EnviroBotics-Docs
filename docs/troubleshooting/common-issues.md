# Troubleshooting

### ❌ My robot doesn't move
- Check motor power supply
- Inspect wiring
- Is your motor driver connected correctly?

### ❌ Sensor not detected
- Use a serial monitor to debug
- Check I2C addresses
- Try replacing the sensor

### ❌ Code won't upload
- Confirm correct COM port
- Press reset button before flashing
